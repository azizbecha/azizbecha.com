### Inline Code

Use backticks (`) to highlight inline code snippets.

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}
```

### Blockquotes
Start your quote by adding `>` at the beginning.
> In my walks, every man I meet is my superior in some way, and in that I learn from him.
